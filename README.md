# Text-to-SQL

A minimal **Text-to-SQL** proof-of-concept that converts natural language questions into SQL queries using Large Language Models (LLMs). This project demonstrates how LLMs can enable non-technical users to query databases using plain English.

---

## Overview

Text-to-SQL is an NLP task where a model translates human language into structured SQL queries. This repository contains a Jupyter notebook showcasing a basic implementation of this concept using an LLM and a sample database schema.
